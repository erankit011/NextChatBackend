# NextChat Backend

Backend server for NextChat real-time chat application.

## Tech Stack

- Node.js
- Express
- MongoDB
- Socket.IO
- JWT Authentication

## Folder Structure

```
server/
├── config/           # Configuration files
├── controllers/      # Route controllers
├── middleware/       # Custom middleware
├── models/           # Database models
├── routes/           # API routes
├── services/         # Business logic
└── server.js         # Main server file
```

## Setup

```bash
npm install
npm start
```

## Development

```bash
npm run dev
```
